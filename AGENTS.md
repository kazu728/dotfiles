# Global instructions

- 回答は日本語で行う。
- 設定・実装はツールのデフォルト挙動を優先する。デフォルトで要件を満たせるなら、独自の style / ラベル / フォーマット等の上書き設定を足さない。
- 既存コードへの継ぎ足し（論理的凝集）で済ませず、触れる範囲は1から設計したような構造で実装する。ただし変更範囲そのものは最小限に保つ。
- 変更は最小限・最小差分にとどめ、要求された範囲を超えて広げない。
- コードレビューは自分（実装したコンテキスト）で結論を出さず、別の fresh subagent に diff だけ渡して実行させる（実装者バイアスの排除）。実装意図や弁明を足さず、検証を通った指摘は握りつぶさない。日常は subagent に /code-review high、徹底時は deep-review。
